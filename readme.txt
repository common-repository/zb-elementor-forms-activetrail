Elementor forms ActiveTrail integration

== Description ==
Elementor Pro forms integration with ActiveTrail. Simply add an After submit action to Elementor Pro forms module. This action post the form data to ActiveTrail group using the ActiveTrail API.

### How to use
https://www.youtube.com/watch?v=DOzFxf9N5f8

= Notes =
* ActiveTrail API token and group id is **required**.
* Email field is **required**.

== Installation ==
= FROM WORDPRESS DASHBOARD =
* Visit 'Plugins > Add New'
* Search for 'ZazimBareshet - Elementor Forms ActiveTrail'
* Activate 'ZazimBareshet - Elementor Forms ActiveTrail' from Plugins page.

= INSTALL MANUALLY =
* Unzip the downloaded zip file.
* Upload the zb-elementor-forms-activetrail folder to the /wp-content/plugins/ directory
* Activate 'ZazimBareshet - Elementor Forms ActiveTrail' from Plugins page.
